Simple Java Programs:
1) Write a program to perform basic arithmetic operations (addition, subtraction, multiplication,
division) using variables and operators.
2) Create a program to convert temperature between Celsius and Fahrenheit using variables and
arithmetic operators.
3) Write a program to check if a given year is a leap year or not using variables, arithmetic
operators, and conditional statements.
4) Develop a program to calculate the factorial of a given number using variables, arithmetic
operators, and loops.
5) Create a program to check if a given string is a palindrome or not using variables, string
operators, and loops.
